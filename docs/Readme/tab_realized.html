Realized tab

Realized is a derived view of realized events (cashflow + FIFO basis) used for reporting. If realized rows don’t match expectations, it generally means FIFO needs rebuilding.

How to use it

  • Filter by User + Site first; then expand to broader reporting.
  • Export CSV for tax/reporting workflows.
  • Use it to cross-check redemption cost basis results.
Realized depends on allocation rebuilds. If it’s stale in Repair Mode, rebuild from Tools.
