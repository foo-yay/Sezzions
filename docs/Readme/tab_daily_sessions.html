Daily Sessions tab

Daily Sessions provides day-level grouping for reporting and review. It’s useful for tying together notes and outcomes for a day.

Common workflow

  1. Filter by date range and user/site where applicable.
  2. Review daily rollups for consistency with purchases/redemptions/sessions.
  3. Export CSV for reporting if needed.

Tips

  • If daily rollups look incorrect after edits/imports, run recalculation.
  • Use this tab as a “sanity check” after large changes.
