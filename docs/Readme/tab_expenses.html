Expenses tab

Expenses are authoritative inputs used for reporting. They generally do not affect FIFO allocations.

Common workflow

  1. Add expense records with date, amount, category/tag, and notes.
  2. Filter and export for reporting.

Tips

  • Use consistent categories/tags so exports are clean.
  • Back up before large imports.
