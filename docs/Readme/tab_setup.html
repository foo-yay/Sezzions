Setup tab

Setup is where you manage the catalog/config entities that the rest of the app depends on. This includes Users, Sites, Game Types, and other reference data.

What to do first on a new DB

  1. Create at least one User.
  2. Create at least one Site.
  3. Configure any game types / categories you want for sessions.

Tools entrypoint

The Tools area is accessed from Setup. See Tools (deep dive) for: repair mode, recalculation, CSV import/export, adjustments, and database utilities.
