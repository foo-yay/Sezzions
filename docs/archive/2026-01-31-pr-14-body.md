## QTreeWidget Technical Challenge

**Qt Limitation:** QTreeWidget doesn't support true cell-level selection. When an item is selected, all columns are selected (Qt architecture constraint).

**Solution:** Single-cell detection via `currentColumn()` and `currentItem()`:
- If exactly one item selected AND it's the current item → treat as single-cell
- Extract only the clicked column's value for statistics
- Multi-selection falls back to full row extraction (all columns)

This provides "feels like cell selection" UX while respecting Qt constraints.
